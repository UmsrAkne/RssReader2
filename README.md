# RSS Reader 2

## キーボード操作説明

フィードリストにフォーカスしている状態で、以下のキーボードによる操作をサポートしています。

| Key | 機能説明                                 |
|-----|--------------------------------------|
| J   | カーソルを一つ下のフィードに移動します。                 |
| K   | カーソルを一つ上のフィードに移動します。                 |
| U   | 選択しているフィードを未読状態に戻します。既に未読の場合は動作しません。 |
